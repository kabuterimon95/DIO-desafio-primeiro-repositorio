-- O Git Bash é um terminal extendido para otimizar o uso do Git.

Comandos gerais, terminal Windows
- cd > navega nas pastas do disco
- dir > lista todos os diretórios e arquivos disponíveis na pasta
- mkdir > cria um novo diretório
- del / rmdir > apaga e remove, diretórios e arquivos

Comandos gerais, terminal Unix
- cd > navega nas pastas do disco
- ls > lista todos os diretórios e arquivos disponíveis na pasta
- mkdir > cria um novo diretório
- rm -rf > apaga e remove, diretórios e arquivos

Comandos terminal Git Bash
- git status > verifica qual a situação do repositório local
- git add * ou -A ou . >  adiciona arquivos/pastas no controle de versão
- git commit -m > coloca mensagem coerente para realizar o commit
- git push origin main > envia todos os "commits" criados localmente para o servidor/nuvem
